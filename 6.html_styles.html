HTML Styles

<style> is a attribute used to add styles to an element, such as color, font, size and more.

Syntax of <style>

style attribute follows the syntax below:

<tagname style="property:value;"

property is a CSS property; value is a CSS value.

The Properties

Here are some properties:

background-color: color of the background, can be set for <body>, <p>, etc.

color: color of the text

font-family: font

font-size: sizze of text

text-align: horizontal alignment of an element
