Basic Elements of HTML Document

What are the basic elements of a HTML document, and what do they mean? This chapter will cover the strcuture element of a HTML document and some other basic tags.

Structure of a HTML Document

<!DOCTYPE html>*

All HTML documents must start with <!DOCTYPE html>.

<!DOCTYPE html> is the declaration defines that this documen is an HTML5 document, always on the first line.

<html> tag

The HTML document begins with <html> and ends with </html>, which are the root element of a HTML page.

<head> & <body> tag

<head> contains the meta information of a HTML page.

<title> is the tile of HTML page, which is shown on the browser's title bar or tab's title's bar.

<body> defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

Other Basic Elements**

<h#> defines heading, there h# range from h1 to h6.

<p> defines paragraph.

<a> defines HTML link. Attribute href specifies the link's destination

<img> defines HTML images. Attribute includes source(src), alternative text(alt), width(width) nad height(height).

*in HTML document '<' is code using (& lt), '>' is code using (& gt).
