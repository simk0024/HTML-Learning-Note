HTML Text Formatting

HTML contains several elements for defining text with special meaning

HTML Formatting Elements

<b> - Bold text. 
	Example: this is bold
	
<strong> - Important text. 
	Example: this is strong
	
<i> - Italic text. 
	Example: this is italic
	
<em> - Emphasized text. 
	Example: this is emphasized
	
<mark> - Marked text. 
	Example: this is marked
	
<small> - Smaller text. 
	Example: this is smaller
	
<del> - Deleted text. 
	Example: this is deleted
	
<ins> - Inserted text. 
	Example: this is inserted
	
<sub> - Subscript text. 
	Example: this is subscript
	
<sup> - Superscript text. 
	Example: this is superscipt
