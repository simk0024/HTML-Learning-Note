HTML Headings

HTML headings define titles and subtitles in a web page.

Types of Heading

Heading range from <h1> to <h6>. <h1> defines the most important heading, <h6> defines the least important heading.

Importance of Heading

Search engines use the headings to index the structure and content of web pages

DO NOT use heading tag to get bold and larger font size, instead please use style tag.

On top of heading default setting, user can use style to define size/font/color of heading
