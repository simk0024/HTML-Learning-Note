HTML Paragraphs

<p> element is used to define a paragraph. A paragraph always starts on a new line and browsers automatically add white space (margin) after 1 paragraph.

Extra spaces and next line in <p> element will be removed when showing in a browser, and displayed as one paragraph.

Horizontal Rules and Line Breaks

<hr> tag defines a thematic break in a HTML page, for example:

This is part 1

This is part 2

<br> tag defines a line break, for example:

This paragraph
is break
into
4 lines.
