HTML Attributes

HTML attributes provide additional information about HTML elements.

All HTML elements can have atrributes, specified in the start tag

Attributes usually come in name/value pair like: name="value"

Common Attributes

Here are some common attributes

href

Attribute of <a>, href attribute specifies the URL of a page.

src

Attribute of <img>, src attribute specifies the path to the image to be displayed.

2 types of URL in src: 1. Absolute URL 2. Relative URL(Recommended)

width and height

Attribute of <img>, width attribute and height attribute specify the size of image in pixel.

alt

Attribute of <img>, alt attribute specifies alternative text of an image.

In case that image cannot be displayed, alt text will be displayed.

style

Attribute of <p> or other elements, style attribute add styles to an element.

lang

Attribute of <html>, lang attribute declares the language of the web page.

Language should always be stated.

title

title attribute defines extra information about an element.

The value of this attribute will be displayed as a tooltip when mouse over.
