HTML Elements

A HTML document comprises of multiple HTML elements. These elements are NOT case sensitive.

How a HTML element looks like?

The HTML element is everthing from the start tag to the end tag, like below:

<tagname> /content/ </tagname>

Nested HTML Elements

HTML elements can be nested - one contains another one.

For example: <head> and <body> are nested in <html>, <title> is nested in <head>, etc.

Empty HTML Elements

Empty HTML Elements are HTML elemtent with no content

<br> tag defines a line break, is an empty HTML element
